## How to Install

1. Download the latest version of Mellel for macOS.  
2. Open the DMG installer and move Mellel.app to the Applications folder.  
3. Launch the app and allow it through macOS Gatekeeper if required.

## Overview

<a href="#">Mellel for macOS</a> is tailored to writers who live inside long documents and care deeply about structure. Instead of optimising for casual notes, it emphasises outlines, styles and reference management. Chapters, sections and subsections can be organised in a clear hierarchy, making it easier to navigate manuscripts, theses or reports. The style system favours consistency and durability, so that formatting changes can be applied cleanly across large bodies of text without breaking layouts.

Academic and technical writers benefit from tools that manage citations, cross-references and complex layouts. Footnotes, endnotes and <a href="#">multi-language support</a> are treated as core features, not afterthoughts.
